## 目标
在现有 `/data/import` 三步导入向导中增加“科目余额表标准化导入”流程，让用户完成字段确认、科目匹配、警告确认并执行入库。

## 交付
1. 数据类型选择中新增：
   - `科目余额表标准化导入`
2. 流程建议拆成以下步骤：
   - 上传文件。
   - 字段映射与年度/期间确认。
   - 金额列模式确认。
   - 科目层级确认。
   - 客户科目到标准科目匹配。
   - 警告确认与执行导入。
3. 字段映射要求：
   - 继续使用稳定 `column_id`。
   - 客户科目代码可选。
   - 客户科目名称建议填写；如果没有名称但有代码，允许进入待映射。
   - 余额方向、科目类别可选。
   - 年度/期间可以由模板默认值或用户手填。
4. 金额映射要求：
   - 支持已有借贷两列直接映射。
   - 支持单列金额按标准科目方向拆分。
   - 支持用户覆盖为借方、贷方、按标准方向。
   - 前端必须明确展示用户当前选择，不允许静默套用。
5. 层级确认要求：
   - 有代码时展示系统按代码推断的层级。
   - 无代码但有缩进时展示“根据 Excel 缩进推断，需确认”。
   - 无代码无缩进时展示平铺明细。
   - 父级默认折叠，展开后显示下一级明细。
6. 科目匹配要求：
   - 展示客户优先候选和全局候选。
   - 历史映射指向停用标准科目时，只显示警告候选，不自动套用。
   - 未映射标准科目的末级行阻止最终入库。
   - 支持用户手动搜索并选择标准科目。
7. 警告确认要求：
   - 父级金额与子级汇总不一致时展示警告，由用户决定是否继续。
   - 负数金额展示警告。
   - 所有阻止项必须清零后才能执行导入。
8. 页面中文可见，无英文裸露文案。

### 完成内容

1. **数据类型选择**：
   - 新增「科目余额表标准化导入」选项（`standardized_trial_balance`）
   - 切换数据类时自动重置步骤和状态，原有三步流程不受影响

2. **导入向导新增 4 个步骤**：

   **步骤 1 — 上传文件：**
   - 上传客户科目余额表 Excel/CSV
   - 填写客户标识（被审计单位名称）、会计年度、会计期间
   - 调用 `POST /standard-trial-balance-imports/preview` 解析表头和样本行

   **步骤 2 — 字段与金额映射：**
   - 字段映射表：将文件列映射到标准字段（科目代码、科目名称、余额方向、科目类别、各类金额列）
   - 金额列拆分模式：单列金额可选择「按标准方向拆分」「全部记入借方」「全部记入贷方」；借贷双列自动识别
   - 层级识别模式：自动/代码前缀/缩进/平铺
   - 年度/期间确认：可从文件列或手动输入获取
   - 调用 `POST /{batch_id}/analyze` 执行字段映射、层级识别、科目映射推荐和金额拆分

   **步骤 3 — 层级确认与科目匹配：**
   - 左侧面板：层级树（行号、科目代码/名称、父级/末级标签、层级来源标识）+ 金额明细预览（六列借贷金额）
   - 右侧面板：逐行科目匹配
     - 展示推荐候选（客户历史 > 全局历史 > 代码匹配 > 名称相似），含来源标签和置信度
     - 停用标准科目的候选显示警告标签，不自动选中
     - 高置信度候选（≥90% 且无警告）自动预选
     - 支持手动搜索标准科目（调用 `/standard-accounts` API）
     - 选中停用科目时弹出提示
   - 阻止项：未映射末级科目行阻止「执行导入」按钮，显示红色错误提示
   - 警告项：展示警告列表（父级不一致、负数金额等）

   **步骤 4 — 执行导入：**
   - 调用 `POST /{batch_id}/execute` 提交确认映射和警告确认
   - 显示执行进度
   - 成功：显示条目数、映射经验保存数
   - 失败：显示中文错误信息
   - 按钮链接跳转到「数据查看」页

3. **金额映射要求**：
   - 借贷双列直接映射：无需额外配置
   - 单列金额按标准方向拆分（`single_by_direction`）
   - 单列金额可覆盖为全部借方（`single_as_debit`）或全部贷方（`single_as_credit`）
   - 前端明确展示当前选择，不静默套用

4. **层级确认要求**：
   - 有代码时按代码前缀推断层级（来源标签：代码）
   - 无代码有缩进时标注「缩进推断」
   - 无代码无缩进时平铺
   - 父级/末级标签明确区分

5. **科目匹配要求**：
   - 客户优先候选 + 全局候选按优先级展示
   - 停用标准科目候选带警告标签，不自动套用
   - 未映射末级行阻止执行
   - 支持手动搜索标准科目

6. **全部中文文案**，无英文裸露。
